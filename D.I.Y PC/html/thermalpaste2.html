Applying New Thermal Paste

Thermal paste can be re-applied easily. If you need help removing the old thermal paste, please refer to this page

Here's what you will need:

-Thermal paste (can be purchased at any computer parts store)


Method:

Put a pea sized blob of thermal paste on the cpu

Once it has been applied, place the fan and heatsink ontop of the CPU and install it. Try to place it correctly in one go.

Make sure not to get any on the motherboard. This is very important.
